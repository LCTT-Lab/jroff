Token

new Token()

Represents a single token, encapsulates common behavior useful to parse and manipulate tokens
Properties:
Name Type Description
value string
kind number of the token, used to know if the token represents a macro, break, inline macro, etc.
nodes array is a collection of sub tokens, useful while parsing ( for example a macro with inline macros ).
